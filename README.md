# **Описание проекта
Конвертер валют позволяет быстро перевести сумму из одной валюты в другую. Пользователь выбирает исходную валюту, целевую валюту и сумму перевода. Конвертация выполняется на основе текущих курсов валют, полученных с сайта Центрального банка России.

# **Требования
Интерпретатор Python версии 3.x.
Библиотека requests для запросов к API ЦБРФ.

# **Инструкция по запуску
Сохраните код в файле currency_converter.py.
Запустите приложение командой:
bash
# **Копировать**
python currency_converter.py
